Manager:

 

The manager screen contains multiple sub-screens and functions. All the functions will be explained.

 

The mainscreen looks as the image below:

 

The functions on this screen include the following:

●       Diagram/table

●       Line diagram button

●       Bar diagram button

●       Table button

●       Print statistics button

●       refresh button

●       Help function

●       Change password function

●       Logout button

 

All the above functions above will be explained in detail.

 

 

 

Diagram/table

 

This area will display the current selected diagram or table as shown below.

 

 

Line diagram button

 

The mainscreen of the manager will always start on the line diagram. The line diagram will show you the lost, found and resolved luggage in a line diagram .

 

When you are on a different diagram/table you can access the line diagram by clicking this button:

 

 

To filter the data in the diagram/table you can use the datepicker to filter on date, or use the checkboxes to filter on what kind of luggage you want to see in the diagram/table.

 

Bar diagram button

 

The bar diagram will show you the lost, found and resolved luggage in a bar diagram.

 

When you are on a different diagram/table you can access the bar diagram by clicking this button:

 

 

To filter the data in the diagram/table you can use the datepicker to filter on date, or use the checkboxes to filter on what kind of luggage you want to see in the diagram/table.

 

Table button

 

The table will show you the lost, found and resolved luggage in a table.

When you are on a different diagram/table you can access the table by clicking this button:

 

 

To filter the data in the diagram/table you can use the datepicker to filter on date, or use the checkboxes to filter on what kind of luggage you want to see in the diagram/table.

 

Print statistics button

 

This function will save the statistics on your computer in a PDF file, you can print the statistics with that file.

 

When you press the button it will bring you to the following screen:

 

 

 

You will need to fill in a name for the file and a location to save the file. When filled in you can press the save button and the file will open and will show the statistics you wanted to print

 

Refresh button

 

When you want to refresh your diagram/table, you can press the refresh button as shown below:

 

When this button is pressed your diagram/table will be refreshed and newly added luggage will be loaded in to your diagram/table.

Help function

 

To call upon the help function press the ? button on the screen as shown below:

You can also acces the help function by pressing the F1 button on your keyboard

 

 

In the help function you can look up information about the function you are using. And additional information to help you work better.

Change password function

 

To call upon the change password function press the change password button on the screen as shown below:

 

 

The change password function will look as shown below:

This function can be used to change your old password to a new password. You can only access this function when you are logged in to the application.

 

Logout button

 

The logout button can be found on the top right of your screen as shown below:

The logout button will log of your account and bring you to the login screen of the application. From here other people can log in or you can use one of your other accounts with different roles.
