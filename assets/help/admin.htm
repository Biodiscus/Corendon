Users table

 

The table you see on the screen will show the current selected information about the users.

 

 

Search user

 

 

The search user function will look as shown below:

 

 

 

 

This function is meant to search through the users in the system.

The fields that you will fill in will be the field that the system will search for.

If a field stays empty the system will not search for that detail. After the field are filled in you can press the Search button to search and the Cancel button to cancel the search.

 

When the system finds any matches the matches will be shown in the users table.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Add user

 

The add user button will bring you to the add user screen as shown below:

 

In this screen you can add a new user, while in this screen you will see multiple fields you have to fill in.

 

Here we have a list of the field and the purpose of the fields:

 

Username fill in the username for the user.

First name fill in the first name for the user.

Last name fill in the last name for the user.

Password fill in the password for the user.

Repeat password repeat the password for the user.

Role fill in the role of the user.

Airport fill in the airport that the user work at.

Contact details fill in any contact details for the user.

Notes a field to add additional information about the user.

 

When all the field are filled out, you can add the user by pressing the Add button. To cancel you can press the Cancel button.

 

When the user is added, the user will be shown in the users table.

 

Edit user

 

 

The edit user button will bring you to the edit user screen as shown below:

 

Username edit the username for the user.

First name edit the first name for the user.

Last name edit the last name for the user.

Password edit the password for the user.

Repeat password repeat the password for the user.

Role edit the role of the user.

Airport edit the airport that the user work at.

Contact details edit any contact details for the user.

Notes edit the additional information about the user.

 

When all the field are filled out, you can edit the user by pressing the Save button. To cancel you can press the Cancel button.

 

When the user is edited, the user will be shown with the new information in the users table.

Revert luggage

 

The revert luggage function will allow you to revert deleted luggage.

When an item is selected in the deleted luggage table you can press the revert luggage button en the luggage will be brought back to its original state.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Delete luggage

 

With the delete luggage function you can permanently delete luggage out of the system.

When an item is selected in the deleted luggage table you can press the delete luggage button en the luggage will be deleted permanently.
