Luggage information table

 

The table you see on the screen will show the current selected information about the luggage.

 

 

Quick filter buttons

 

The quick filter buttons named: found, lost and resolved, are meant to be used for quick filtering for the luggage information table. With the buttons you can switch between found, lost and resolved with a single press.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Advanced search

 

The advanced search function will look as shown below:

 

 

This function is meant to search through the luggage that is added into the system.

The fields that you will fill in will be the field that the system will search for.

If a field stays empty the system will not search for that detail. After the field are filled in you can press the Search button to search and the Cancel button to cancel the search.

 

When the system finds any matches the matches will be shown in the luggage information table.

 

 

 

 

 

 

 

 

 

 

 

 

Add luggage

 

The add luggage button will bring you to the add luggage screen as shown below:

 

 

In this screen you can add new luggage, while in this screen you will see multiple fields you have to fill in.

 

Here we have a list of the field and the purpose of the fields:

 

Lost/found the choose if you want to add lost luggage or found luggage.

Found on airport select the airport the luggage is found, in case it is lost leave this empty.

Label fill in the label number of the luggage, if unknown leave this field empty.

Color select the main color of the luggage.

Picture upload a picture of the luggage if available, when uploaded the picture will

show in the white box on the right of the screen.

Brand fill in the brand of the luggage.

Dimensions fill in the height, width and depth of the luggage.

Weight fill in the weight of the luggage in kilograms.

Notes a field to add additional information about the luggage.

 

 

When all the field are filled out, you can add the luggage by pressing the Add button. To cancel you can press the Cancel button.

 

When the luggage is added, the luggage will be shown in the luggage information table.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Edit luggage

 

The edit luggage button will bring you to the edit luggage screen as shown below:

 

 

Found on airport edit the airport the luggage is found, in case it is lost leave this empty.

Label edit the label number of the luggage, if unknown leave this field empty.

Color edit the main color of the luggage.

Picture upload a picture of the luggage if available, when uploaded the picture will

show in the white box on the right of the screen.

Brand edit the brand of the luggage.

Dimensions edit the height, width and depth of the luggage.

Weight edit the weight of the luggage in kilograms.

Notes edit additional information about the luggage.

 

 

 

When all the field are filled out, you can add the luggage by pressing the Save button. To cancel you can press the Cancel button.

 

When the luggage is edited, the luggage will be shown with the new information in the luggage information table.

Resolved

 

The button resolved as shown below:

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Will bring you to the following screen:

 

 

Here you have to fill in the following information of the associated customer:

 

First name fill in the first name from the associated customer.

Last name fill in the last name from the associated customer.

Address fill in the address from the associated customer.

Zipcode fill in the zipcode from the associated customer.

State fill in the state from the associated customer.

Country fill in the country from the associated customer.

E-mail fill in the e-mail from the associated customer.

Phone fill in the phone number from the associated customer.

 

 

 

When all the field are filled you can press the Print and set resolved button.

That will allow you to save the PDF on your computer and print the file. The luggage status will be set to resolved.

 

 

 

 

 

 

Print details

 

 

The button Print details will bring you to a new screen.

 

 

 

 

 

 

 

On this screen you can select the location you want to save the file containing the details about the luggage. After filling in the name you can save the file(PDF) and select it and then print the file in a PDF viewer.

 

 

 

 

 

 

 

 

Delete luggage

 

When an item is selected in the luggage information table the delete luggage button can be pressed to delete a item in the table.

 

 

 

 

When you want to recover a deleted item you can contact your administrator. This person can recover deleted luggage.
